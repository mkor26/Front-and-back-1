# Интерент магазин электронники на React + Express

### Бэкенд
- Node.js
- Express
- Nanoid
- Swagger

### Фронтенд
- React
- Axios
- SCSS

### Как запустить проект
Скачайте проект, перейдите в терминале в папку "backend" (cd backend) и выполните команду (npm install), далее сделайте то же самое с папкой "frontend" ((cd ..) (cd frontend) (npm install)).
Откройте терминал в папке backend и frontend и запустите по отдельности (npm start).

### Документация API
Swagger UI доступен по адресу: http://localhost:3000/api-docs
Документация содержит:
- Схему товара (Product)
- Все эндпоинты с описаниями
- Интерактивное тестирование API
